# configs

用于存放 detector、stabilizer、scene、arbitration、behavior、runtime 等配置文件。

目录约定：
- `configs/runtime/local/`：本地机 profile
- `configs/runtime/desktop_4090/`：4090 / CUDA profile
- `configs/detectors/local/`：本地 detector profile
- `configs/detectors/desktop_4090/`：4090 detector profile
- 其他如 `arbitration/`、`stabilizer/`、`safety/`、`slow_scene/` 继续按职责集中维护
